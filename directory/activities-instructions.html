Activities

    Activities is a feature that allows you to plan and share activities with people in your quorum, class, calling or other group. It has three main sections.

  • Lets plan something to start planning an event.
  • Upcoming to easily view all upcoming activities.
  • And Activity Ideas to help with coming up with ideas for activities

Planning an Activity

  • Select Let's plan something.

  • Type in a name, location, date, and time, and add details in the purpose section.

  • Invite the intended group by selecting Invite Circles. Add any other additional information such as a photo or add link.

  • Select Save in the top right corner to save your activity.

  • If you invited any Circles to your activity, it will automatically put your activity on their page and they will be able to RSVP.

Event Details

  • To see event details, set a notifications, and to edit, start by selecting the event.

  • To add a reminder select the reminder option under the Notification header.

  • To Edit, Duplicate, Add to calender or Remove, select the the dot icon in the top right. (This option is only available if you organized the event)

RSVP to an activity

When you are invited to an activity through Circles it will appear under the Upcoming tab.

Use the dropdown menu next to the event to indicate whether you can attend.
